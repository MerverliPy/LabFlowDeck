# /resume-phase

Resume work on the current phase in `.opencode/plans/current-phase.md`.

## Required behavior

- Re-read the active phase file.
- Summarize what is already complete versus still pending.
- Continue only the remaining in-scope tasks.
- Preserve the existing candidate ID and acceptance criteria.
- Do not select a new phase unless explicitly instructed.

## Expected output

Report:
- candidate ID
- remaining tasks
- files likely to change next
- validation still required
