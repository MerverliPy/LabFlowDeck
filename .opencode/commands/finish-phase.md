# /finish-phase

Finalize the active phase after implementation and validation.

## Required behavior

- Check the phase goal and acceptance criteria.
- Update `.opencode/plans/current-phase.md` with final status and summary.
- Record the validation command and result.
- Prepare the phase for `/ship-phase`.

## Expected output

Report:
- final status
- acceptance criteria outcome
- validation result
- completion summary
- recommended next phase trigger
